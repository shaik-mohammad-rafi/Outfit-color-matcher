# Outfit-color-matcher
The Outfit Color Matcher detects the top’s color (typed or uploaded) and instantly suggests matching bottom colors with direct shopping links.
Outfit Color Matcher

The Outfit Color Matcher is a lightweight, ML-based tool that identifies the dominant color of a user's top—either from a typed color or an uploaded image—and recommends matching bottom colors. The system also provides direct shopping links from platforms like Flipkart, Amazon, Meesho, Myntra, and Ajio for quick and convenient outfit selection.

Features:
Detects dominant colors from uploaded images
Accepts color input through text
Suggests matching outfit combinations
Provides shopping links for recommended colors
Simple, clean, and user-friendly interface

Tech Stack:
Python, Flask
HTML, CSS, JavaScript
OpenCV, sklearn
Kotlin (Android app version)
